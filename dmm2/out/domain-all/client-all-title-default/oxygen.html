How do I connect Oxygen XML Editor to Kuali KMS?

Connecting Oxygen XML Editor to Kuali KMS through WebDAV

You must have Oxygen XML Editor 11 (Windows, Mac) installed on your computer.
Oxygen XML Editor can connect to Kuali KMS using WebDAV. This allows you to load content from the KMS into Oxygen, perform authoring tasks, and then save your changes directly to the KMS.
  1. Open Oxygen XML Editor.
  2. Select Project > New Project . In the Project view, newProject.xpr appears.
  3. Right-click newProject.xpr, and then choose Import Remote Folders.
  4. In Open using FTP/SFTP/WebDAV:
    1. In Identification, enter any username and password. Neither field can be empty. Check Save.
      Note: Do not use your IU Network ID passphrase.
    2. Next to Server URL:, enter http://cowhorn.uits.indiana.edu:8080/sage-stg/repository/default/ and check Autoconnect.
  5. Click Browse.
  6. In the file list, click default, and then click OK.
  7. In the Project view, double-click newProject.xpr to reveal the files from the KMS. You can now open, edit, and save files on the KMS through the Project view.
Choose Project > Save Project to save the project on your computer.
Note: The project file on your computer contains the KMS connection information, so you will not have to repeat these steps if you save the project.
